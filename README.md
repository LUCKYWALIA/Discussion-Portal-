Discussion Portal

This is a lightweight, fully client-side (HTML, CSS, JavaScript) question and answer application. It allows users to post questions with a subject, view question details, add responses, and sort/filter the list of questions based on a search term or the number of responses (popularity).

This application uses plain JavaScript Maps for data storage, all questions and responses are lost when the browser tab is closed or the page is refreshed.
